How is the staging area different from the working directory and the repository? What value do you think it offers?
staging area is pre-commit place. To be able to decide which changes to commit.

How can you use the staging area to make sure you have one commit per logical change?
Only add the changes you want to commit by using git add. 
You can double check the change you want to commit by using git diff --staged
